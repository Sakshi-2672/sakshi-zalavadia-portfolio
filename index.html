PROJECTS

Twirfy - One-to-one Video Calling Web App

Twirfy is an one-to-one video calling web application which allows two people to join in an virtual room and share their live media like audio and video. In this project ReactJS is used to develop attractive front-end and NodeJs is used for back-end. And WebRTC is used for the real time communication.

Bon Voyage - Tour Booking Management System

Bon Voyage is an user-friendly web application for easily booking the dream travel destination and also book the additional adventures and sty in hotels of user's choice. In this project HTML & CSS is used to develop front-end and PHP is used for back-end and to stoare all the user data MySQL database is used.

Connect 4 Game

The Connect 4 Game is two player game where player need to connect 4 consecutive dots either horizontally, vertically or diagonally. This game can also be played with the single player and AI. In the development of the game PyGame library is used. It is an set of Python modules used to design video games.

Home Automation System

The main objective of this project is to develop a home automation system using an Arduino UNO board with Bluetooth being remotely controlled by an Android OS smartphone. In this project Arduino UNO and HC-05 Bluetooth module is used. This application can be operated using a smartphone and bluetooth connection.
